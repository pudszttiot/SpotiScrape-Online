About Us

SpotiScrape is a web application that allows users to extract information from Spotify URLs. Whether it's a track or a playlist, SpotiScrape provides you with detailed information about the content, including artist, album, track name, and artwork.

Key Features:

  • Extract information from Spotify track URLs
  • Extract information from Spotify playlist URLs
  • Display detailed information about the content, including artist, album, and track/playlist name
  • Download artwork images associated with the track or playlist
  • Clear input to start over

SpotiScrape is a useful tool for music enthusiasts, playlist creators, and anyone curious about the details of their favorite tracks and playlists on Spotify.
